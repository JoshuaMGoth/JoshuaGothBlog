A Shut-Down Prompt That Can Save Us All

I have memory problems. As one that suffers from CRS (cant remember sh!t), anything that I can use to aid my handicap from day-to-day, I am all for. As such, I forget to save things and end up losing work on a daily basis during the evenings shutdown of the PC.

There are countless sites with walkthroughs, guides, programs, etc, so I do my best as to not reinvent the wheel, and share things that I have found useful that there may not have had enough attention to gain traction.

But It’s the little tech-shortcuts that I aim to share in my tech-centered posts. Things that, as a real person—who makes tons of real mistakes—I find really useful.

It took some time and testing, but I created a script that I would like to share that is made for folks such as myself. The primary purpose: Saving all unsaved documents in a folder when a shutdown is triggered. Built in with a simple prompt, I have worked this into a service, which allows for the saving of all unsaved documents before any shutdown automatically.

I hope you find it as useful as I have. Just update the variables to suit your personal needs Don’t forget the test run. 1

Auto-Save Script

First things first: Create a script called autosave_work.sh and paste the contents below. I created two folders on my desktop. One for where the unsaved (now-saved) work goes, and one where the scripts that you will be creating below. But to each, their own.

  1#!/bin/bash
  2
  3# Auto-save work script
  4# Save to: ~/Desktop/Shutdown
  5
  6AUTOSAVE_DIR="$HOME/Desktop/Shutdown"
  7TIMESTAMP=$(date +"%Y%m%d_%H%M%S")
  8BACKUP_DIR="$AUTOSAVE_DIR/work_backup_$TIMESTAMP"
  9
 10# Create backup directory
 11mkdir -p "$BACKUP_DIR"
 12
 13echo "Auto-saving work to: $BACKUP_DIR"
 14
 15# Function to check if process is running and then save relevant files
 16save_work() {
 17    # Save browser tabs (if using Firefox/Chrome)
 18    if pgrep -x "firefox" > /dev/null; then
 19        echo "Saving Firefox session..."
 20        # Firefox session backup (requires session manager extension or copy profile)
 21        firefox_profile=$(find ~/.mozilla/firefox -name "*.default-release" 2>/dev/null | head -1)
 22        if [ -n "$firefox_profile" ]; then
 23            mkdir -p "$BACKUP_DIR/firefox"
 24            cp -r "$firefox_profile/sessionstore-backups" "$BACKUP_DIR/firefox/" 2>/dev/null
 25        fi
 26    fi
 27
 28    if pgrep -x "chrome" > /dev/null || pgrep -x "chromium" > /dev/null; then
 29        echo "Saving Chrome/Chromium session..."
 30        # Chrome session backup
 31        chrome_dir=$(find ~/.config -name "Chromium" -o -name "google-chrome" 2>/dev/null | head -1)
 32        if [ -n "$chrome_dir" ]; then
 33            mkdir -p "$BACKUP_DIR/chrome"
 34            cp "$chrome_dir/Default/Current Session" "$BACKUP_DIR/chrome/" 2>/dev/null
 35            cp "$chrome_dir/Default/Current Tabs" "$BACKUP_DIR/chrome/" 2>/dev/null
 36        fi
 37    fi
 38
 39    # Save terminal sessions (tmux)
 40    if pgrep -x "tmux" > /dev/null; then
 41        echo "Saving tmux sessions..."
 42        mkdir -p "$BACKUP_DIR/tmux"
 43        tmux list-sessions > "$BACKUP_DIR/tmux/sessions.txt"
 44        # Save individual pane content
 45        tmux list-panes -a -F "#{session_name}:#{window_index}.#{pane_index}:#{pane_current_command}:#{pane_current_path}" > "$BACKUP_DIR/tmux/panes.txt"
 46    fi
 47
 48    # Save unsaved documents from common editors
 49    save_editor_work() {
 50        local editor=$1
 51        local process_name=$2
 52        
 53        if pgrep -x "$process_name" > /dev/null; then
 54            echo "Detected $editor running - attempting to save work..."
 55            case $editor in
 56                "code")
 57                    # VS Code - save all files
 58                    if command -v code >/dev/null 2>&1; then
 59                        code --list-extensions > "$BACKUP_DIR/vscode_extensions.txt"
 60                        # Save opened files list
 61                        code --get-file-watch-patterns > "$BACKUP_DIR/vscode_files.txt" 2>/dev/null
 62                    fi
 63                    ;;
 64                "vim"|"nvim")
 65                    # Vim/Neovim - save session
 66                    mkdir -p "$BACKUP_DIR/vim"
 67                    # You might want to manually save sessions in your vim config
 68                    find /tmp -name "*vim*" -type f 2>/dev/null | head -5 | xargs -I {} cp {} "$BACKUP_DIR/vim/" 2>/dev/null
 69                    ;;
 70            esac
 71        fi
 72    }
 73
 74    save_editor_work "code" "code"
 75    save_editor_work "vim" "vim"
 76    save_editor_work "nvim" "nvim"
 77
 78    # Save recent documents
 79    echo "Saving recent documents..."
 80    find ~/Documents -name "*.txt" -o -name "*.md" -o -name "*.pdf" -o -name "*.doc*" 2>/dev/null | \
 81        head -20 | xargs -I {} cp {} "$BACKUP_DIR/" 2>/dev/null
 82
 83    # Save desktop files
 84    echo "Saving desktop files..."
 85    cp ~/Desktop/* "$BACKUP_DIR/" 2>/dev/null
 86
 87    # Save clipboard content
 88    if command -v xclip >/dev/null 2>&1; then
 89        echo "Saving clipboard content..."
 90        xclip -selection clipboard -o > "$BACKUP_DIR/clipboard.txt" 2>/dev/null
 91    fi
 92
 93    # Save open windows information
 94    if command -v wmctrl >/dev/null 2>&1; then
 95        echo "Saving window information..."
 96        wmctrl -l > "$BACKUP_DIR/open_windows.txt"
 97    fi
 98}
 99
100# Export function for use in subshells
101export -f save_work
102export BACKUP_DIR
103
104# Run save function
105save_work
106
107# Create a summary file
108echo "Auto-save completed at: $(date)" > "$BACKUP_DIR/summary.txt"
109echo "Backup location: $BACKUP_DIR" >> "$BACKUP_DIR/summary.txt"
110echo "User: $USER" >> "$BACKUP_DIR/summary.txt"
111
112echo "Auto-save completed! Files saved to: $BACKUP_DIR"

Make it Executable

1chmod +x autosave_work.sh

Enhanced Safe Shutdown Script with Interactive Prompt

Create a file calledinteractive_shutdown.sh, paste the contents below and place it with the other scripts.

  1#!/bin/bash
  2
  3# Interactive safe shutdown script with auto-save
  4
  5AUTOSAVE_SCRIPT="./autosave_work.sh"  # Update path if needed
  6
  7# Function to display usage
  8show_usage() {
  9    echo "Usage: $0 [minutes|HH:MM]"
 10    echo "Examples:"
 11    echo "  $0           - Interactive mode"
 12    echo "  $0 5         - Shutdown in 5 minutes"
 13    echo "  $0 18:30     - Shutdown at 6:30 PM"
 14    echo "  $0 now       - Shutdown immediately"
 15    exit 1
 16}
 17
 18# Function to validate time input
 19validate_time() {
 20    local input=$1
 21    
 22    # Check for "now"
 23    if [[ "$input" == "now" ]]; then
 24        return 0
 25    fi
 26    
 27    # Check for minutes (numeric)
 28    if [[ "$input" =~ ^[0-9]+$ ]]; then
 29        if [ "$input" -le 0 ]; then
 30            echo "Error: Minutes must be greater than 0"
 31            return 1
 32        fi
 33        return 0
 34    fi
 35    
 36    # Check for HH:MM format
 37    if [[ "$input" =~ ^([0-1]?[0-9]|2[0-3]):[0-5][0-9]$ ]]; then
 38        return 0
 39    fi
 40    
 41    echo "Error: Invalid time format. Use minutes (5) or time (18:30) or 'now'"
 42    return 1
 43}
 44
 45# Function to calculate seconds until target time
 46calculate_seconds() {
 47    local input=$1
 48    
 49    if [[ "$input" == "now" ]]; then
 50        echo 0
 51        return
 52    fi
 53    
 54    if [[ "$input" =~ ^[0-9]+$ ]]; then
 55        # Minutes input
 56        echo $((input * 60))
 57        return
 58    fi
 59    
 60    # HH:MM format
 61    local target_hour=$(echo "$input" | cut -d: -f1)
 62    local target_minute=$(echo "$input" | cut -d: -f2)
 63    local current_hour=$(date +%H)
 64    local current_minute=$(date +%M)
 65    local current_second=$(date +%S)
 66    
 67    local target_total_minutes=$((target_hour * 60 + target_minute))
 68    local current_total_minutes=$((current_hour * 60 + current_minute))
 69    
 70    local minutes_diff=$((target_total_minutes - current_total_minutes))
 71    
 72    # If target time is earlier, assume it's for tomorrow
 73    if [ $minutes_diff -lt 0 ]; then
 74        minutes_diff=$((minutes_diff + 24 * 60))
 75    fi
 76    
 77    local seconds_diff=$((minutes_diff * 60 - current_second))
 78    echo $seconds_diff
 79}
 80
 81# Function to run auto-save
 82run_autosave() {
 83    echo "Starting auto-save procedure..."
 84    
 85    if [ -f "$AUTOSAVE_SCRIPT" ]; then
 86        bash "$AUTOSAVE_SCRIPT"
 87        if [ $? -eq 0 ]; then
 88            echo "✓ Auto-save completed successfully"
 89            return 0
 90        else
 91            echo "⚠ Auto-save encountered issues"
 92            return 1
 93        fi
 94    else
 95        echo "❌ Auto-save script not found at: $AUTOSAVE_SCRIPT"
 96        return 1
 97    fi
 98}
 99
100# Function for interactive prompt
101interactive_prompt() {
102    echo "=========================================="
103    echo "    Safe System Shutdown with Auto-Save"
104    echo "=========================================="
105    echo
106    echo "How would you like to schedule the shutdown?"
107    echo
108    echo "1) Shutdown in X minutes"
109    echo "2) Shutdown at specific time (HH:MM)"
110    echo "3) Shutdown now"
111    echo "4) Cancel"
112    echo
113    read -p "Enter your choice [1-4]: " choice
114    
115    case $choice in
116        1)
117            read -p "Enter minutes until shutdown: " minutes
118            if validate_time "$minutes"; then
119                SHUTDOWN_DELAY_SECONDS=$(calculate_seconds "$minutes")
120                SCHEDULE_TYPE="minutes"
121                SCHEDULE_VALUE="$minutes"
122            else
123                echo "Invalid input. Please try again."
124                interactive_prompt
125            fi
126            ;;
127        2)
128            read -p "Enter shutdown time (HH:MM, 24-hour format): " shutdowntime
129            if validate_time "$shutdowntime"; then
130                SHUTDOWN_DELAY_SECONDS=$(calculate_seconds "$shutdowntime")
131                SCHEDULE_TYPE="time"
132                SCHEDULE_VALUE="$shutdowntime"
133            else
134                echo "Invalid time format. Please try again."
135                interactive_prompt
136            fi
137            ;;
138        3)
139            SHUTDOWN_DELAY_SECONDS=0
140            SCHEDULE_TYPE="now"
141            SCHEDULE_VALUE="now"
142            ;;
143        4)
144            echo "Shutdown cancelled."
145            exit 0
146            ;;
147        *)
148            echo "Invalid choice. Please try again."
149            interactive_prompt
150            ;;
151    esac
152}
153
154# Function to format time remaining
155format_time_remaining() {
156    local seconds=$1
157    local hours=$((seconds / 3600))
158    local minutes=$(( (seconds % 3600) / 60 ))
159    local secs=$((seconds % 60))
160    
161    if [ $hours -gt 0 ]; then
162        printf "%dh %dm %ds" $hours $minutes $secs
163    elif [ $minutes -gt 0 ]; then
164        printf "%dm %ds" $minutes $secs
165    else
166        printf "%ds" $secs
167    fi
168}
169
170# Function to display shutdown info
171display_shutdown_info() {
172    local seconds=$1
173    local type=$2
174    local value=$3
175    
176    echo
177    echo "=== Shutdown Scheduled ==="
178    case $type in
179        "minutes")
180            echo "Time until shutdown: $value minutes"
181            ;;
182        "time")
183            echo "Shutdown at: $value"
184            local current_epoch=$(date +%s)
185            local target_epoch=$((current_epoch + seconds))
186            echo "Which is: $(date -d "@$target_epoch" "+%H:%M:%S")"
187            ;;
188        "now")
189            echo "Shutting down immediately"
190            ;;
191    esac
192    echo "Auto-save will be performed"
193    echo "==========================="
194    echo
195}
196
197# Main script logic
198
199# If arguments provided, use them
200if [ $# -eq 1 ]; then
201    if ! validate_time "$1"; then
202        show_usage
203    fi
204    SHUTDOWN_DELAY_SECONDS=$(calculate_seconds "$1")
205    
206    # Determine schedule type for display
207    if [[ "$1" == "now" ]]; then
208        SCHEDULE_TYPE="now"
209        SCHEDULE_VALUE="now"
210    elif [[ "$1" =~ : ]]; then
211        SCHEDULE_TYPE="time"
212        SCHEDULE_VALUE="$1"
213    else
214        SCHEDULE_TYPE="minutes"
215        SCHEDULE_VALUE="$1"
216    fi
217else
218    # Interactive mode
219    interactive_prompt
220fi
221
222# Confirm shutdown
223display_shutdown_info "$SHUTDOWN_DELAY_SECONDS" "$SCHEDULE_TYPE" "$SCHEDULE_VALUE"
224
225if [ $SHUTDOWN_DELAY_SECONDS -gt 10 ]; then
226    read -p "Proceed with shutdown? (y/N): " confirm
227    if [[ ! "$confirm" =~ ^[Yy]$ ]]; then
228        echo "Shutdown cancelled."
229        exit 0
230    fi
231else
232    echo "Shutting down in 5 seconds... (Ctrl+C to cancel)"
233    sleep 5
234fi
235
236# Run auto-save
237if ! run_autosave; then
238    read -p "Auto-save had issues. Continue with shutdown? (y/N): " continue_anyway
239    if [[ ! "$continue_anyway" =~ ^[Yy]$ ]]; then
240        echo "Shutdown cancelled due to auto-save issues."
241        exit 1
242    fi
243fi
244
245# Notify user
246if command -v notify-send >/dev/null 2>&1; then
247    if [ $SHUTDOWN_DELAY_SECONDS -eq 0 ]; then
248        notify-send "System Shutdown" "Shutting down now. Work saved to Desktop/Shutdown/"
249    elif [ $SHUTDOWN_DELAY_SECONDS -le 300 ]; then  # 5 minutes or less
250        notify-send "System Shutdown" "Shutting down in $(format_time_remaining $SHUTDOWN_DELAY_SECONDS). Work saved to Desktop/Shutdown/"
251    else
252        notify-send "System Shutdown" "Shutdown scheduled. Work saved to Desktop/Shutdown/"
253    fi
254fi
255
256# If immediate shutdown
257if [ $SHUTDOWN_DELAY_SECONDS -eq 0 ]; then
258    echo "Shutting down now..."
259    sudo shutdown -h now
260    exit 0
261fi
262
263# Countdown for longer delays
264if [ $SHUTDOWN_DELAY_SECONDS -le 300 ]; then  # 5 minutes or less
265    echo "Final countdown:"
266    while [ $SHUTDOWN_DELAY_SECONDS -gt 0 ]; do
267        if [ $SHUTDOWN_DELAY_SECONDS -le 60 ] && [ $((SHUTDOWN_DELAY_SECONDS % 10)) -eq 0 ] || [ $SHUTDOWN_DELAY_SECONDS -le 10 ]; then
268            echo "Shutting down in $(format_time_remaining $SHUTDOWN_DELAY_SECONDS)..."
269        fi
270        sleep 1
271        SHUTDOWN_DELAY_SECONDS=$((SHUTDOWN_DELAY_SECONDS - 1))
272    done
273else
274    # For longer delays, check every minute
275    local minutes_remaining=$((SHUTDOWN_DELAY_SECONDS / 60))
276    echo "Shutdown in progress... (Ctrl+C to cancel)"
277    
278    while [ $minutes_remaining -gt 0 ]; do
279        if [ $minutes_remaining -le 10 ] || [ $((minutes_remaining % 15)) -eq 0 ]; then
280            echo "$minutes_remaining minutes remaining until shutdown..."
281            
282            if command -v notify-send >/dev/null 2>&1 && [ $minutes_remaining -le 5 ]; then
283                notify-send "Shutdown Reminder" "$minutes_remaining minutes remaining"
284            fi
285        fi
286        sleep 60
287        minutes_remaining=$((minutes_remaining - 1))
288    done
289    
290    # Final minute countdown
291    echo "Final countdown:"
292    for i in {60..1}; do
293        if [ $i -le 10 ] || [ $((i % 15)) -eq 0 ]; then
294            echo "$i seconds remaining..."
295        fi
296        sleep 1
297    done
298fi
299
300echo "Shutting down now..."
301echo "Last auto-save location: ~/Desktop/Shutdown/work_backup_$(date +"%Y%m%d_%H%M%S")"
302
303# Execute shutdown
304sudo shutdown -h now
305#!/bin/bash
306
307# ... [previous script content remains the same] ...
308
309# Function to check sudo privileges
310check_sudo() {
311    echo "Checking for shutdown privileges..."
312    if sudo -n true 2>/dev/null; then
313        return 0
314    else
315        return 1
316    fi
317}
318
319# Function to execute shutdown with proper privileges
320execute_shutdown() {
321    echo "Shutting down now..."
322    echo "Last auto-save location: ~/Desktop/Shutdown/work_backup_$(date +"%Y%m%d_%H%M%S")"
323    
324    # Try to shutdown without password if possible, otherwise prompt
325    if sudo -n shutdown -h now 2>/dev/null; then
326        exit 0
327    else
328        echo "Root privileges required for shutdown. Please enter your password:"
329        sudo shutdown -h now
330    fi
331}

(Optional) Disable Sudo for Shutdowns: For Those Who Love an Easy Life

Don’t worry, we are approaching the end. Next step, edit the sudoers file

1sudo visudo

Add this line (replace yourusername with your actual username—duh)

1yourusername ALL=(ALL) NOPASSWD: /usr/bin/shutdown

Taking Lazy to the Next Level: Creating a systemd service that runs your script when the computer shuts down

Create Service File

1sudo nano /etc/systemd/system/autosave-shutdown.service

Add the following

 1[Unit]
 2Description=Auto-save Work Before Shutdown
 3DefaultDependencies=false
 4Before=shutdown.target reboot.target halt.target
 5
 6[Service]
 7Type=oneshot
 8RemainAfterExit=true
 9ExecStart=/bin/true
10ExecStop=/home/yourusername/path/to/autosave_work.sh
11TimeoutStopSec=30
12User=yourusername
13Environment=HOME=/home/yourusername
14
15[Install]
16WantedBy=multi-user.target

Enable the Service

1sudo systemctl enable autosave-shutdown.service

And Finally, Test the Service

1sudo systemctl start autosave-shutdown.service
2sudo systemctl status autosave-shutdown.service

Taking it to the Next Level: Adding the Power Button for Events

For those who dare venture on: Create acpi(d) script power-button-auto-save

1sudo nano /etc/acpi/events/power-button-auto-save

Paste the Following

1event=button/power.*
2action=/home/yourusername/path/to/autosave_work.sh && /usr/bin/systemctl poweroff

Kick the Daemon (Restart acpi(d))

1sudo systemctl restart acpid

AND FINALLY, Last but Not Least. . .

For those of use who are too lazy to get out of bed to shut the PC down and prefer to SSH into your system by phone

1ssh -t username@hostname 'cd /path/to/scripts && sudo ./interactive_shutdown.sh'
