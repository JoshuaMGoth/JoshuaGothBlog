## How to Use

1. **Upload an image** by dragging & dropping or clicking the upload area
2. **Adjust the settings** to customize your ASCII art
3. **Export your creation** as text or HTML
4. **Share with family and friends**

## Features

- **Multiple character sets** for different artistic effects
- **Adjustable density and contrast** for optimal results
- **Color modes** including monochrome, grayscale, and color preservation
- **Privacy-focused** - all processing happens in your browser

## Tips for Best Results

- Use high-contrast images for clearer ASCII art (.SVG / No backgrounds work the best)
- Try different character sets for unique effects
- Adjust the character density for more or less detail
- Simple graphics work best for color preservation mode

## Technical Details

- **Maximum file size**: 10MB
- **Supported formats**: JPG, PNG, GIF, WebP
- **Browser requirements**: Modern browser with JavaScript enabled
- **Works on**: Desktop and mobile devices
